# ji v1.0.0

## Summary

Initial release. Encrypt dotfiles into a single `.ji` file, sync via WebDAV, manage multi-device access with recipient keys.

## What's Included

### Core
- `ji init` — configuration and age keypair generation
- `ji add / rm / list / status / diff` — local manifest management
- `ji pack / unpack` — encrypt/decrypt .ji archives
- `ji check` — integrity verification (fast + deep)
- `ji doctor` — config, keys, manifest, and connectivity diagnosis

### Remote
- `ji remote add / remove / list / test / files / delete` — WebDAV endpoint management
- `ji push / pull / sync` — remote transfer
- SSH remote via feature flag (`--features ssh`)

### Security
- `ji recipient list / add / remove` — multi-recipient key management
- PGP encryption via feature flag (`--features pgp`)

### Developer
- `ji completion` — shell completions (bash/zsh/fish)
- `.jiignore` — gitignore-style exclusion rules

## Install

```bash
cargo install --path .
```

## Upgrade Notes

First release, no upgrade path needed.

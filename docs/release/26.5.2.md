## Fixes

- **`ji pack` now updates manifest checksums.** After packing, `ji status` correctly shows "(no changes)" instead of stale "M" markers. The manifest and `.ji` file stay in sync.
- **`ji status --short` only shows changed files.** Previously it dumped all tracked files including unchanged ones. Now both default and `--short` modes only show changes â€” `--short` suppresses headers for scripting.
- **`ji add` re-tracking.** Running `ji add` on an already-tracked file now refreshes its checksum. If the file hasn't changed, it's a silent no-op.
- **`ji check --deep` returns error on mismatch.** Previously it printed `CHECKSUM MISMATCH` but still said `deep check: OK` with exit 0. Now it collects all mismatches, reports them, and exits 1.
- **`ji rm --all` no longer requires a PATH argument.** You can now run `ji rm --all` directly.
